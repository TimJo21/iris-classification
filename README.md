# iris-classification

## Objective
End-to-end machine learning pipeline for multi-class classification using Iris dataset.

---

## Architecture
- Data module (train/test split)
- Training module (RandomForest)
- Evaluation module (metrics + confusion matrix)
- Visualization module (feature importance)

---

## Tech Stack
- Python
- Scikit-learn
- Matplotlib
- Joblib

---

## How to Run

```bash
pip install -r requirements.txt
python iris_classification.py
